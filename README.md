## Config guidelines:

### For Windows: 
```bash
cd %USERPROFILE%/AppData/Local
git clone https://github.com/dannyHallo/nvim.git
```

### For MacOS: 
```bash
cd ~/.config/
git clone https://github.com/dannyHallo/nvim.git

# or directly:
mkdir -p ~/.config/nvim && git clone https://github.com/dannyHallo/nvim.git ~/.config/nvim
```
